# 가장 쉽게 시작하는 법

## 1. 서버 세팅

클로드 코드에게 이렇게 말하면 됩니다:

```
SERVER_RUN_GUIDE.md 읽고 그대로 세팅해줘
```

## 2. AI 세팅

AI 기능(코멘트 필터링, 원인 분석 등)을 쓰려면 AI 프로바이더를 설정해야 합니다.

```
AI_SETUP_GUIDE.md 읽고 내 컴퓨터에 맞게 AI 세팅해줘
```

또는 직접 설정하려면 [AI_SETUP_GUIDE.md](./AI_SETUP_GUIDE.md)를 읽어보세요.
GPU 사양에 따라 로컬(Ollama) vs 클라우드(API) 중 어떤 게 맞는지 안내되어 있습니다.
